-------------------------------------------------------
---	            Math Library Tester                 ---
-------------------------------------------------------

Welcome to the MathLibrary Tester project, designed to provide some hard-coded tests of the 
functionality of the MathLibrary to ensure that the results given are correct.

This Readme will serve as a changelog, as well as document the individual sections of the application.

---------------------
- CHANGELOG	        -
---------------------

v0.6 (10-11-2009):
Small update for output of Vector Normalise() - wrong variable used for display.

v0.5 (20-10-2009):
Contains tests for all functions currently implemented in the Vector and Matrix classes of the MathLibrary.
	- Currently missing from MathLibrary: Matrix equality.
